# 🌍 World Bank Data Analyzer (worldbank.py)

Программа для анализа данных Всемирного банка. Загружает CSV-файл с показателями по странам, вычисляет статистику за указанный год, строит график распределения значений и выделяет страны с аномальными показателями.

---

## 📥 Требования

- Python 3.7+
- Библиотеки:
  - `matplotlib`
  - `csv` (встроенная)
  - `sys` (встроенная)

Установка matplotlib:
```bash
pip install matplotlib
