# IFS Fractal Generator

Генератор фракталов методом систем итерационных функций (IFS).

## Запуск

```bash
python ifs.py <config_file> <output_image>
```

## Пример

```bash
python ifs.py fern.png
```

## Формат конфигурации

Первая строка: вероятности и название.
Далее — строки с 6 коэффициентами `a b c d e f`.

## Алгоритм

1. Начать с (0,0)
2. На каждом шаге выбрать преобразование по вероятности
3. Применить: `x = a*x + b*y + e`, `y = c*x + d*y + f`
4. Отбросить первые 100 точек
5. Сохранить результат в PNG

## Зависимости

- matplotlib

Установка: `pip install matplotlib`
