# 🌬️ Wind Rose Visualizer (rosewind.py)

Программа для построения **розы ветров** на основе метеорологических данных.  
Анализирует CSV-файл с данными о направлении ветра (`wdir`) за 2025 год (г. Минск) и строит диаграмму, показывающую преобладающие направления ветра.

---

## 📥 Требования

- Python 3.7+
- Библиотеки:
  - `matplotlib`
  - `csv` (встроенная)
  - `sys` (встроенная)
  - `math` (встроенная)

pip install matplotlib
