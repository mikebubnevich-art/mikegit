# 📊 Sorting Algorithms Comparison (sorting.py)

Программа, реализующая три **«медленных»** алгоритма сортировки:  
- Пузырьковая сортировка (bubble sort)
- Сортировка выбором (selection sort)
- Сортировка вставками (insertion sort)

Для каждого алгоритма подсчитывается количество **сравнений** и **обменов** элементов.

---

## 📥 Требования

- Python 3.7+

Библиотеки не требуются (используется только стандартный модуль `sys`).
